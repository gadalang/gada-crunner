C runner for [gada](https://github.com/gadalang/gada).

## Install

Using pip:

```bash
pip install gada-crunner
```

## Testing

The `test` directory contains many tests that you can run with:

```python
python setup.py test
```

Or with coverage:

```python
coverage run --source=gada_crunner setup.py test
```
